Componentes Bootstrap

En este proyecto podras encontras algunas de las diferentes funciones que se pueden utilizar con bootstrap.

Accordion

Un acordeón es un conjunto de encabezados interactivos apilados horizontal o verticalmente que muestran y ocultan paneles plegables, cada panel contiene una sección del contenido.

Ver
Alerts

Las Alerts, son un tipo de notificacion o aviso el cual se utiliza para avisar al usuario sobre algo importante, las alerts pueden anunciar, aprovacion, peligro, cuidado, etc.

Ver
Notifications

Son pequeños iconos o indicadores mediante los cuales se le informa algo al usuario

Ver
Information

Tienen como objetivo informar al usario sobre la paginación y la posicion en el sitio

Ver
Buttons

Es un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite.

Ver
Card

Un Card es un apartado en el cual se pueden agregar multiples objetos como lo son imagenes, botones, links, textos, etc.

Ver
Carousel

Un Carousel es una espacion el el cual se puede presentar diversa informacion de manera vistosa para el usuario asi como lo pueden ser imagenes.

Ver
Collapse

El Collapse es un elemento plegable que se muestra luego de precionar un espacio determinado, usualmente un boton.

Ver
Dropdowns

Un Dropdown es un elemento o lista que se sobrepone para mostar listas y enlases.

Ver
List group

El list groups es un componente de utilidad ya que permite mostar informacion en fomar de lista lo cual facilita la vista para el usuario.

Ver
Modal

El Modal una funcion en forma de cuadro de dialogo emergente, una de sus funciones es el uso de Notificacíon

Ver
Navs and tabs

Los navs son una seccion utilizada para proporcionar enlaces de navegacion al usuario.

Los tabs son un recurso utilizado paa organizar los contenidos web

Ver
Navbar

La Navbar es un elemento de la interfaz del usuario dentro de una página web que contiene enlaces a otras secciones del sitio web.

Ver
Offcanvas

Un offcanvas es una barra lateral oculta que se hará visible cuando se dispare un evento. Muy útil en dispositivos móviles donde tenemos poco espacio en pantalla ya que facilitara la navegacion entre las paginas de un sitio web.

Ver
Placeholder

El atributo de marcador de posición especifica una pequeña pista que describe el valor esperado de un campo de entrada (por ejemplo, un valor de muestra o una breve descripción del formato esperado).

Ver
Popovers

El popover es un tipo de cuadro emegentes que surge al enviar un evento como precionar un boton, funciona para notificar o avisar algo al usuario.

Ver
Progress

Los Progress son barras de carga que su pueden utilizar para mostrañe al usuario el avance de un item, componente, etc.

Ver
Scrollspy

Scrollspy es un complemento de arranque que actualiza automáticamente los enlaces en una lista de navegación según la posición de desplazamiento actual.

Ver
Toasts

El componente Toast es como un cuadro de alerta que solo se muestra durante un par de segundos cuando sucede algo (es decir, cuando el usuario hace clic en un botón, envía un formulario, etc.).

Ver
Tooltips

Tooltip es un pequeño cuadro emergente que aparece cuando el usuario mueve el puntero del mouse sobre un elemento

Ver
Tables

Las tables le permiten agregar una gran cantidad de datos y presentarlos de manera clara y ordenada para facilidad del usuario

Ver
Forms

Un formulario es un conjunto de controles (botones, cajas de texto, casillas de verificación, botones radio, etc) que permiten al usuario introducir datos y enviarlos al servidor web para su procesamiento.
